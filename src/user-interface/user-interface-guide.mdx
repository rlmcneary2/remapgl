## Attribution

The [Attribution](/attribution) component provides information about the creators and owners of
the map. One or more strings can be added to the attribution using the `customAttribution` prop.
Attribution is a child of the [Map](/map) component.

```jsx
<Map accessToken={accessToken}>
  <Attribution customAttribution="remapgl<ðŸŒŽ>" />
</Map>
```

## Geolocate

The [Geolocate](/geolocate) control will move the map to the user's current location. Geolocate is a child of the [Map](/map)
component.

```jsx
<Map accessToken={accessToken}>
  <Geolocate />
</Map>
```

## Navigation

The user may zoom the map or reorient it to North using the
[Navigation](/navigation) component. Navigation is a child of the [Map](/map)
component.

```jsx
<Map accessToken={accessToken}>
  <Navigation />
</Map>
```

## Scale

The [Scale](/scale) control displays the ratio of a distance on the map to the
corresponding distance on the ground.

```jsx
<Map accessToken={accessToken}>
  <Scale maxWidth={200} unit="imperial" />
</Map>
```
