Map

The <Map/> component creates and displays a map. The Map can be controlled and interacted with via its props. The only required prop is accessToken. This is a string token provided by Mapbox, you will need to create a Mapbox account to get one.

A Map component can accept a number of other components as children. Data at a specific point is displayed on a map using a Marker. Information that applies to paths or regions on a map can be applied by Layer components.

Sample

<Map
  accessToken="my_token"
  center={[-68.8008887, 44.5591077]}
  motionType="ease"
  zoom={9}
>
  {
    markers.map(marker => (
      <Marker
        location={marker.location}
      >{marker.name}</Marker>
    ))
  }
  <Layer
    id={selectedLayer.id}
    paint={selectedLayerPaint}
    source={selectedLayer.source}
    type="path"
  />
  {
    layers.filter(layer => layer.id !== selectedLayer.id)
      .map(layer => (
        <Layer
          id={layer.id}
          paint={layerPaint}
          source={layer.source}
          type="path"
        />
      ))
  }
</Map>

PropertyTypeComments
centerLngLatThe geographical center point of the map. Changing this value will cause the map position to change.

LngLat

Represents a geographic location on the map. When using the tuple (array) signature the first element is the longitude and the second element is the latitude. [number, number] | { lat: number; lng: number; }

LngLatBounds

Represents a geographic rectangle on a map. A tuple; the first element is the south west corner, the second element is the north east corner. [LngLat, LngLat]
